# Build Instructions
### Linux

```
git clone https://github.com/Spacerulerwill/Minecraft-Clone.git --recursive
cd "Minecraft-Clone/tools/unix"
./configure.sh
./build.sh
```

### Windows
```
git clone https://github.com/Spacerulerwill/Minecraft-Clone.git --recursive
cd "Minecraft-Clone\\tools\\windows"
./configure.bat
./build.bat
```
